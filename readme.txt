== Description ==

This is a multipurpose slider plugin.

**Demo :** <a href='http://webzotech.com/awesomelite/' target='_blank'>http://webzotech.com/awesomelite/</a>

<a href='http://www.webzotech.com/awesomeslider' target='_blank'>Upgrade to Pro Version</a>

<a href='http://www.support.webzotech.com/' target='_blank'>Premium Support</a>

It comes with 3 level device control where different settings can be applied for different devices. Disable element or change element property on a particular device. So the slider is properly visible and readable in all devices.

Each element is draggable and resizable with aspect ratio for easy placement. Which gives the freedom to instantly see what you are doing, and no knowledge of codes is required, and furthermore you are not bound down by any redesigned templates, complete freedom in a visual-based environment.

**Powerful Layer Manager :** You do not need calculator to calculate appearance time and easing time. Awesome Slider gives an option to manage the animation timings through an interface. So it can be managed in a visual environment too.

**Inbuilt CSS Editor :** Awesome slider is built with CSS editor with autocomplete option. Which gives the freedom to write css code in the browser like an advanced CSS editor.

== Installation ==

1. Upload the entire `awesomesliderlite` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Awe Slides' and 'Awe Sliders' menu in your WordPress admin panel.

== Frequently Asked Questions ==

= How to create a slide? =

Navigate to "Awe Slides" and click on "Add New Slide" button.
Once Slide create click on "Manage" link. 

= How to create a slider? =

Navigate to "Awe Slider" and click on "Create New Slider".
Enter Slider name and click on "Add New". 

= How to use this? =

Just put the short code [awslider id=xxx]

= How to make the slide responsive? =

There are three level device control options. All the elements can be re-arranged in different devices.

= Can I disable any device? =

Yes.
